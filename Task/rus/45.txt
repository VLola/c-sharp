Проект 45.

Написать свой провайдер-контактов.  Приложение должно выводить список контактов. Должен быть функционал :
- Добавление контакта
- Удаление контакта
- Поиск по имени/фамилии/отчеству/оператору/дате добавления
- Сортировать по имени/фамилии/отчеству/оператору/дате добавления
- При клике на карточку - должно открываться окно с более детальной инфой

Карточка состоит с:
- Имя *
- Фамилия
- Отчество
- Дата рождения
- Номер телефона (1*-10) с автоопределением оператора
- Адрес (страна / город / улица / дом / квартира)
- Машина (марка / модель / рег номер) 
- Компания (имя / должность)

* обязательные поля
